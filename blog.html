QUESTIONs & ANSWERs

Question 1: What is HTML?
HTML stands for HyperText Markup Language. It is a standard markup language used to create and design web pages and web applications.
Question 2: What are the basic elements of HTML?
The basic elements of HTML are tags, attributes, and values. Tags are used to define the structure of the content, attributes are used to provide additional information about the tags, and values are used to set the content within the tags.
Question 3: What is the difference between HTML and CSS?
HTML is used to structure and define the content of a web page, while CSS is used to style and layout the content of a web page. HTML focuses on the content, while CSS focuses on the presentation of the content.
Question 4: What is the purpose of the tag in HTML?
The tag is used to provide information about the document such as the title of the document, links to external files, and metadata about the document such as keywords and descriptions. It is not displayed on the web page.
Question 5: What is the purpose of the tag in HTML?
The tag is used to define the main content of the web page. This is where you would include text, images, videos, and other media that you want to display on the web page.
